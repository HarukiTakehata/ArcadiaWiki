# 开始使用 Open Parties and Claims

本文将帮助你在 theArcadia 整合包的多人游戏中快速上手 OPAC 模组。

---

## 基本按键

| 操作 | 默认按键 | 说明 |
|:-----|:---------|:-----|
| 打开 OPAC 菜单 | `'`（单引号） | 打开主菜单，查看领地、队伍、个人配置等 |
| 声明/取消声明领地 | 通过菜单操作 | 进入菜单后选择相应选项 |

> 按键可在 **游戏设置 → 控制** 中修改。

---

## 声明领地

### 方法一：通过 UI 菜单

1. 按 `'` 键打开 OPAC 主菜单
2. 选择「领地管理」（Claims Manager）
3. 在小地图或菜单界面上选择区块进行声明
4. 声明成功后，区块边界会显示高亮

### 方法二：通过聊天指令

```
/openpac-claims add
```

> 如果你安装了 [Xaero 的世界地图](https://modrinth.com/mod/xaeros-world-map)，可以在世界地图界面上**右键拖拽选择区域**来批量声明领地。

---

## 领地权限管理

1. 按 `'` 打开 OPAC 菜单
2. 选择「领地管理」（Claims Manager）
3. 选择一个已声明的区块
4. 在弹出的菜单中调整各项权限：

| 权限 | 说明 |
|:-----|:------|
| 破坏方块 | 是否允许破坏方块 |
| 放置方块 | 是否允许放置方块 |
| 打开容器 | 是否允许打开箱子、熔炉等 |
| 与实体交互 | 是否允许与村民、动物等交互 |
| 使用红石物品 | 是否允许操作红石相关物品 |

每个权限可设置为：**所有人** / **仅队伍** / **仅自己**。

## 队伍系统

### 创建队伍

```
/openpac-parties create <队伍名称>
```

### 队伍管理指令

| 指令 | 说明 |
|:-----|:------|
| `/party invite <玩家名>` | 邀请玩家加入队伍 |
| `/party kick <玩家名>` | 踢出队伍成员 |
| `/party leave` | 离开当前队伍 |
| `/party info` | 查看队伍信息 |
| `/opm <消息>` | 发送队伍聊天消息 |

### 队伍聊天

使用 `/opm <消息>` 快速向队伍成员发送消息，或使用 `O` 键切换队伍聊天频道。

---

## 个人配置

通过 OPAC 菜单或使用指令 `/openpac` 可以配置个人设置：

- **最大声明数量**：可声明的区块总数上限（由服务器配置）
- **强制加载限制**：可强制加载的区块数量上限
- **区块互动权限默认值**：新声明的区块默认权限

> 部分选项可能被服务器管理员锁定，灰色显示即表示不可修改。

---

## 实用技巧

- 🏠 **建造基地前先声明领地**：防止未授权玩家破坏
- 👥 **和朋友一起游戏时建立队伍**：自动共享领地权限
- 🗺️ **使用 Xaero 地图**：在地图上直观查看和管理所有声明区块
- ⚙️ **调整领地例外**：在领地中开放特定权限（如允许他人使用箱子商店）来促进交易
- 🚫 **避免过度声明**：注意服务器的声明数量限制
