## 概述

**theArcadia** 是一个**类原版**（Vanilla-friendly）的 Minecraft 整合包，旨在保有原版游戏精神的前提下，极大地丰富游戏体验。

有了 Arcadia，你能享受到和原版 Minecraft 相近的游戏体验：**没有固定任务，没有既定目标，没有多少限制**——纯粹的沙盒自由。

> ⚠️ **整合包目前处于活跃开发阶段**，新内容正在持续添加中。

## 内容构成

我们精心挑选并整合了超过 **100 个模组**，涵盖六大类别：

- **[Supplementaries](https://modrinth.com/mod/supplementaries)** — 装饰与功能小物件

## 客户端体验

theArcadia 整合了 **30+ 个客户端模组**，力求精致入微的体验：

### 🎨 UI 与排版

- **ModernUI** — 矢量字体渲染，CJK 完美，缩放无锯齿
- **FrontierUI** — 统一美观的 GUI 界面
- **FancyMenu + Drippy Loading Screen** — 启动画面不再是 2011 年的进度条

### ℹ️ 信息显示

- **Jade** — 指向即知方块信息
- **EMI + JEI** — 全配方查询
- **AppleSkin** — 食物饱食度内联显示
- **Traveler's Titles** — 跨群系/维度时显示史诗级标题
- **PickUpNotifier** — 自动记录背包变化

### 🎭 角色动画

- **Not Enough Animations** — 攀爬、爬行、抓梯动作更自然
- **3D Skin Layers** — 帽子、盔甲有层次感
- **Chat Animation** — 聊天消息平滑弹出动画

### ⚡ 性能优化

- **Sodium + Iris** — 光影高帧率流畅运行
- **ModernFix + FerriteCore** — 减少 GC 卡顿，区块加载平滑
- **动态光源** — 无额外性能开销
